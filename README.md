# 起点小说后端
## 技术栈
本次使用的技术栈主要有ssm框架，filter过滤器，jwt令牌，阿里云oss存储图片，数据库mysql
## 实现功能
- 用户注册
- 用户登录
- 用户个人信息修改(头像,昵称等)
- 主页搜索
  - 通过关键字
  - 通过其分类进行查询
- 小说详情
  - 基本信息(作者信息, 所在分类, 作品简介等）
